## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.

## Structure

- `_posts/` - Blog posts
- `projects/` - Project directories
- `about.md` - About page
- `projects.md` - Projects page
- `index.md` - Home page
- `_config.yml` - Jekyll configuration
